Learning path

Your structured 18-week roadmap

Four phases, each ending with a real portfolio project reviewed by Maria personally.

PHASE 1 · WEEKS 1–3

Python & Programming Foundations

Beginner · No prior experience needed

  • Python syntax, data types & control flow
  • Functions, modules & OOP fundamentals
  • File I/O, error handling & debugging
  • Git version control & CLI basics
  • Mini project: command-line task manager
PHASE 2 · WEEKS 4–8

Web Backend: Django & PostgreSQL

Intermediate · MVC architecture & databases

  • Django models, views & URL routing
  • REST APIs with Django REST Framework
  • PostgreSQL & query optimization
  • Authentication, authorization & security
  • Project: full blog API with user auth
PHASE 3 · WEEKS 9–13

Frontend: HTML, CSS, JS & React

Intermediate · Building interactive UIs

  • HTML5 semantics & CSS3 / Tailwind
  • JavaScript ES6+: async/await, fetch API
  • React components, hooks & state
  • Connecting frontend to your Django API
  • Project: React analytics dashboard
PHASE 4 · WEEKS 14–18

Capstone: Full Stack App + Job Readiness

Advanced · Ship & present a real product

  • Full stack Django + React e-commerce app
  • Stripe payment & third-party integrations
  • Testing with pytest & end-to-end Selenium
  • Deployment on Render / Railway / Heroku
  • Portfolio review & interview preparation
